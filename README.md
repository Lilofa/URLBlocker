# URL Blocker

Простая программа для блокировки сайтов на Windows.

## Как работает

Программа добавляет правила в брандмауэр Windows. Заблокированный сайт не открывается ни в браузере, ни в других приложениях.

## Как использовать

1. Запустите `URL Blocker.exe` от имени администратора
2. Нажмите `START`
3. Введите домен (например, `youtube.com`) и нажмите `ADD`
4. Сайт заблокирован

Чтобы отключить блокировку — нажмите `STOP`

## Кнопки

- `START` — включить блокировку
- `STOP` — выключить блокировку
- `ADD` — добавить сайт в список
- `REMOVE` — удалить сайт из списка

## Что можно вводить

- `youtube.com`
- `https://youtube.com`
- `https://youtube.com/watch?v=123`

Любой вариант подойдет.

## Требования

- Windows 10 или Windows 11
- Запуск от имени администратора

## Установка

Никакой установки не требуется. Просто запустите `.exe` файл.

## Удаление

Просто удалите файл `URL Blocker.exe`.

## Вопросы и ответы

**Почему программа не запускается?**  
Запустите от имени администратора (правой кнопкой мыши → "Запуск от имени администратора").

**Почему сайт не блокируется?**  
Убедитесь, что нажата кнопка `START`.

**Программа собирает данные?**  
Нет. Программа работает локально и не подключается к интернету.
